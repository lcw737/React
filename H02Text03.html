문자열 묶을때 큰따옴표

JAVA에서 문자열은 큰따옴표로 감싼다. "문자열" 형태로 문자열 사용함.
큰따옴표는 "로, 작은따옴표는 '로 표현한다.
단, '는 IE(인터넷익스플로러)에서 사용이 불가능하다.

소스코드를 그대로 웹브라우저에 표시

public class HelloWorld
{
    public static void main(String[] args)
    {
        System.out.println("안녕하세요");
        System.out.println("HelloWorld..!!");
        System.out.println("굿~모닝입니다.");
    }
}
